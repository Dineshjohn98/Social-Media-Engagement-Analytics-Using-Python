# Social Media Engagement Analytics

## 📌 Project Overview

This project focuses on analyzing social media engagement data using Python for Data Analytics. The objective is to uncover meaningful insights related to user engagement, content performance, audience behavior, sentiment analysis, and platform usage patterns through data cleaning, statistical analysis, and visualization techniques. The project leverages powerful Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn to transform raw social media data into actionable business insights. 

---

## 🎯 Objectives

* Perform data cleaning and preprocessing on social media engagement data.
* Analyze user engagement metrics such as likes, comments, shares, and watch time.
* Identify the best-performing content categories and post types.
* Explore audience demographics and behavior patterns.
* Visualize trends and correlations using various charts and plots.
* Generate actionable insights to improve social media strategies.

## 📊 Key Insights

### Content Performance

* **Text posts** achieved the highest engagement rate (**191.5**).
* **Travel** emerged as the best-performing content category.

### Geographic Analysis

* **Brazil** recorded the highest average engagement rate (**1.57**).

### Audience Behavior

* Users aged **50–55** showed the highest engagement.
* Users aged **60–65** showed the lowest engagement.

### User Verification

* A significant proportion of users held verified status.

### Device Usage

* **Mobile devices** contributed the highest watch-time share (**33.6%**).

### Sentiment Analysis

* **Positive Sentiment:** 47.1%
* **Neutral Sentiment:** 30.6%
* **Negative Sentiment:** 19.3%

The analysis indicates a predominantly positive user experience, while reducing negative sentiment remains an opportunity for platform improvement.
